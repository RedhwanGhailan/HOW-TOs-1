HOW-TO Setup Cloudstack
-----------------------

1. Import the VM into VBox (DevCloud2_421.ova)

2. Set-up VBox Host-Only network (vboxnet0) - disable DHCP
    should be on the 192.168.56.1

3. Ensure the NIC #1 of the VM is configured on the correct Host-Only

4. SSH into the box (or use the VBox console)::

    ssh root@192.168.56.10
    
5. Build cloudstack::

    cd cloudstack
    export MAVEN_OPTS="-Xmx1024m -XX:MaxPermSize=800m -Xmx1g -Xdebug -Xrunjdwp:transport=dt_socket,address=8787,server=y,suspend=n"
    mvn -P developer,systemvm clean install
    
6. Initialize the DB with some default values::

    mvn -P developer -pl developer,tools/devcloud -Ddeploydb

The settings can be seen in the SQL script::

    cat developer/developer-prefill.sql 
    
7. Run the Jetty server:

    mvn -pl :cloud-client-ui jetty:run

Then login to the management console: http://192.168.56.10/client
(``admin``/``password``)

8. Update the IP of the management server in **Global Setting** search for the
``host`` key and change it to ``192.168.56.10``

Terminate the server (Ctrl-C) and restart it::

    mvn -pl :cloud-client-ui jetty:run

9. Deploy some basic resources::

    cd ~/cloudstack/tools/devcloud
    python ../marvin/marvin/deployDataCenter.py -i devcloud.cfg

10. RESULT!
    You should now see some basic resources available and be able to deploy a VM
